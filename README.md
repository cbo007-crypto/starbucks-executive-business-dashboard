
# Starbucks Executive Business Dashboard

## Project Overview
This project analyzes Starbucks' financial performance from FY2022–FY2025 using publicly available annual reports.

## Tools
- Microsoft Excel

## Skills Demonstrated
- Financial Analysis
- Business Analytics
- KPI Development
- Dashboard Design
- Data Visualization

## Key Deliverables
- Executive Dashboard
- Financial KPI Analysis
- Segment Analysis
- Strategic Recommendations

## Data Source
Starbucks Investor Relations Annual Reports
